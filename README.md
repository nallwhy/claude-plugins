# claude-plugins

A collection of [Claude Code](https://docs.anthropic.com/en/docs/claude-code) plugins by nallwhy.

## Plugins

### Elixir

Skills for Elixir development with Claude Code.

**Install:**

```
claude plugin add nallwhy/claude-plugins/plugins/elixir
```

**Skills:**

| Skill | Description |
|---|---|
| `/elixir:elixir-pkg-setup [packages...]` | Install and configure Elixir packages. Follows official guides, handles package-specific setup (ash, tidewave, usage_rules), and verifies compilation. |
| `/elixir:elixir-ai-dev-setup` | Set up AI-assisted development tools (usage_rules, tidewave). |
